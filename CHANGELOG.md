## 版本说明

### 版本号格式
`主版本号.次版本号.修订号`

- **主版本号**：重大功能变更或不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新类型
- ✨ **新增功能** (Added) - 新增的功能
- 🔄 **功能变更** (Changed) - 对现有功能的变更
- 🗑️ **废弃功能** (Deprecated) - 即将移除的功能
- ❌ **移除功能** (Removed) - 已移除的功能
- 🐛 **Bug 修复** (Fixed) - 任何 bug 的修复
- 🔒 **安全修复** (Security) - 安全相关的修复
